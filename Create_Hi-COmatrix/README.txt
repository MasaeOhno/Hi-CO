1) In "main.vi", input names of the Read1 and Read2 files (.sam) to "SAM file path 1" and "SAM file path 2".
2) In "get nucleosome positions.vi", input a list of nucleosome positions to "nucleosome positions (chr)" and "nucleosome positions (base)", and select "Edit/Make current values default".
3) In "get chromosome locus info.vi", input a list of nucleosome numbers in every chromosome to "nucleosome number", and a list of base pair length of every chromosome to "chromosome length base", and select "Edit/Make current values default".
4) Run.

Screenshots of the codes are provided in the "png" folder.
